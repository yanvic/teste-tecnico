## Requisitos do Ambiente

Certifique-se de ter o Node.js instalado em sua máquina. Este projeto requer a versão 14 do Node.js.

```bash
# Verificar a versão do Node.js
node -v

# Instalar o Node.js versão 14 node v14.21.1

npm run dev
